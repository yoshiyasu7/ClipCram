# ClipCram

[🇺🇸EN](README.md) |  🇷🇺 RU

Утилита для эффективного сжатия видеофайлов без визуальной потери качества. Проект разработан для оптимизации хранения медиафайлов и .

## 🚀 Статус проекта

- **Текущий этап**: Инициализация проекта (Initial Commit).
- **Цель**: Создать быстрый и удобный инструмент для уменьшения размера видео с сохранением исходного разрешения и четкости.

## 🛠 Стек технологий

- Python / FastAPI
- FFmpeg
- Web UI
- Celery
- Redis
- aioboto3 (S3-storage)
- SQLite
- pytest
- structlog
